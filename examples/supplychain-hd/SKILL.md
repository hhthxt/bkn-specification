# HD 供应链业务知识网络 - Agent 使用指南

> **网络ID**: supplychain  
> **版本**: 1.0.0  
> **标签**: 供应链, 采购, 库存, MRP

## 网络概览

本知识网络描述产品供应分析完整路径：需求预测→产品→BOM→库存→生产计划/采购申请→采购订单→供应商。

### 业务逻辑

1. 基于需求预测，结合 BOM 结构展开物料需求
2. 核查成品及物料库存，进行 MRP 运算生成建议订单
3. 根据物料属性（自制/外购），生成生产计划或采购申请
4. 采购申请转化为采购订单并由供应商执行，生产计划指导工厂生产
5. 最终输出缺口数量及补货建议，支撑供应链决策

### 核心对象

| 对象类别 | 对象列表 |
|----------|----------|
| 需求侧 | 需求预测、产品需求计划 |
| 产品侧 | 产品、产品BOM |
| 物料侧 | 物料、库存、物料需求计划 |
| 计划侧 | 工厂生产计划 |
| 采购侧 | 采购申请单、采购订单、供应商 |
| 销售侧 | 销售订单 |

### 核心关系

| 关系 | 说明 |
|------|------|
| 需求→产品 | 需求预测关联产品 |
| 产品→BOM | 产品展开物料清单 |
| BOM→物料 | 物料需求计算 |
| 物料→库存 | 库存核查 |
| 库存→MRP | 生成建议订单 |
| MRP→计划/采购 | 区分自制/外购 |
| 采购申请→订单 | 采购执行 |
| 订单→供应商 | 供应商履约 |

## 拓扑结构

```
需求预测 → 产品 → BOM → 物料 → 库存 → MRP运算
                                      ↓
                          ┌──────────┴──────────┐
                          ↓                     ↓
                    工厂生产计划            采购申请单
                          ↓                     ↓
                    生产执行               采购订单
                                                ↓
                                           供应商履约
```

## 使用建议

### 查询场景

1. **获取所有对象定义**
   - 读取 `object_types/objects.bkn`
   - 包含 12 个供应链相关对象

2. **查找关系定义**
   - 读取 `relation_types/relations.bkn`
   - 包含 14 个业务关系

### 分析场景

1. **供应缺口分析**
   - 结合需求预测、库存、在途订单
   - 计算净需求并生成补货建议

2. **供应商绩效评估**
   - 分析采购订单履约情况
   - 评估交付及时率和质量
