# BKN SDK

Official SDKs for parsing, validating, and transforming [BKN (Business Knowledge Network)](../docs/bkn_docs/SPECIFICATION.md) files.

## Available SDKs

| Language | Status | Path |
|----------|--------|------|
| **Python** | Available | [sdk/python/](python/) |
| **Golang** | Planned | [sdk/golang/](golang/) |

## Python SDK

Parse `.bkn` files into structured models and transform them to platform-specific formats (e.g., kweaver JSON).

```bash
cd sdk/python
pip install -e .
```

See [python/README.md](python/README.md) for usage and API docs.
